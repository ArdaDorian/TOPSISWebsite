Editable Criteria and Alternative Names

All Criteria and Alternative names can be editable. By doing this, you can prevenet any confusion which can be occur in the future.

Sum of Weights have to be 1

All values in column of weights have to be sum of 1. It cannot be less or more than 1.

Criteria Values have to Be in Same Unit

Each row of criteria, values must be in same unit. If criteria 1 is represents weight all values in that row must be same with first value's unit (all kg or g)
